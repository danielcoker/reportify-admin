# Nuxt 3 Admin Template

Reportify admin dashboard.

## Features

- Login
- View list of reports
- Resolve reports

## Installation

```
# Clone this repository
git clone https://github.com/danielcoker/reportify-admin.git

# Go to directory
cd reportify-admin

# Run with npm
npm install
npm run dev

# Or with pnpm
pnpm install --shamefully-hoist
pnpm dev

# Install Husky
npx husky install
```
